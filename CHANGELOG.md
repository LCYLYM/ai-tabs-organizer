# Changelog

## 0.0.0.1 - 2026-03-17

- Initial Chrome Web Store candidate build.
- Added OpenAI-compatible provider support with customizable `baseUrl`, `apiKey`, and `model`.
- Added Chrome built-in AI provider support via Prompt API and offscreen document workflow.
- Added auto-tagging when tabs lose focus, plus one-click scan for the current window.
- Added rebuild and clear actions for grouped tabs and classification records.
- Added bilingual UI support for Simplified Chinese, English, and follow-browser mode.
- Added classification history, diagnostics, and improved provider test feedback.
- Added optional reclassification for already-grouped tabs when their URL changes.
- Improved content extraction to prioritize visible page text, headings, and main content blocks before broader fallback extraction.
- Added Chrome Web Store packaging workflow and release assets preparation support.
